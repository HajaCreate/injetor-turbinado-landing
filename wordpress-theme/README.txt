=== Curso Reparação Unidade Injetora Theme ===

Descrição: Tema WordPress personalizado para curso de reparação de unidades injetoras
Versão: 1.0
Requer WordPress: 5.0 ou superior
Requer PHP: 7.4 ou superior

== INSTALAÇÃO ==

1. Faça upload da pasta 'wordpress-theme' para '/wp-content/themes/'
2. Renomeie a pasta para 'curso-unidade-injetora'
3. Ative o tema no painel do WordPress
4. Configure as informações de contato em Aparência > Personalizar

== CONFIGURAÇÃO ==

1. Vá em Aparência > Personalizar > Informações de Contato
2. Configure:
   - Número do WhatsApp (formato: 5511999999999)
   - Número do Telefone (formato: 5511999999999)
   - Email de contato

3. Faça upload das imagens na pasta '/images/':
   - hero-image.jpg (1920x1080px)
   - units/ (pasta com imagens das unidades)
   - favicon.ico
   - og-image.jpg (para redes sociais)
